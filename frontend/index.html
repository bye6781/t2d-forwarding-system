📋 如何获取 API ID 和 API Hash？ ▼
  1. 打开 Telegram 官网 my.telegram.org
  2. 输入你的 Telegram 手机号（含国际区号），点击 下一步
  3. 打开 Telegram 客户端，找到来自 Telegram 官方的验证消息，将验证码填入网页
  4. 登录成功后，点击 "API development tools"
  5. 填写应用信息（可随意填写）：
    • App title：随便取个名字，如 MyT2DApp
    • Short name：简短名称，如 myt2dapp
    • Platform：选 Desktop
    • Description：随便写，如 T2D forwarding
  6. 点击 "Create application"，页面会显示你的 api_id 和 api_hash
  7. 将 api_id 和 api_hash 分别填入上方表单即可
⚠️ 每个 Telegram 账号只需创建一次应用，之后 api_id 和 api_hash 可以重复使用。
群组 ID 通常为负数，如 -1001234567890
